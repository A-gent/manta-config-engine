# dotakeract
Dota 2 configuration builder.

## Installation
- Install [node.js](https://nodejs.org)
- Clone or download this repository.
- Execute `node build`.

## Advanced Build
- Execute `node build preset-name`, where `preset-name` is the name of the preset you want to build.
