# Manta Config Engine
Dota 2 configuration builder. Compiles `JSON` to `autoexec`.

## Installation
- Install [node.js](https://nodejs.org)
- Run `npm install -g dota2-manta-config-engine`.
- Execute `manta-config-engine path-to-preset`, where `path-to-preset` is a relative path.

Beware that the CLI will assume that your Dota 2 is installed in the default Windows Steam location. Create a pull request if you can fix this behavior.
